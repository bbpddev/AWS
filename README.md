# AWS

Study material and practice tools for AWS certifications.

## Contents

### AWS Certified Developer – Associate (DVA-C02)

**[[`index.html`](index.html)](https://bbpddev.github.io/AWS/)** — an interactive, self-contained practice exam.
Open the file in any browser; there are no dependencies and no network calls.

- **110 scenario-based questions written in the real DVA-C02 format** — long problem
  stems with requirement qualifiers (LEAST operational overhead, MOST secure, FEWEST
  changes) and four plausible, fully-worked solution options, modeled on the style of
  the actual exam and the major practice providers (Tutorials Dojo, Stephane Maarek).
- Includes **capacity-math questions** (RCU/WCU calculations, transactional read costs)
  and heavy coverage of the topics that dominate the live exam: Lambda (function URLs,
  /tmp encryption, secrets, CloudFormation deployments), SAM CLI (build / local invoke /
  deploy --guided / sync), API Gateway (Regional endpoints, canary releases, custom
  integrations), AppSync (subscriptions, pipeline resolvers), Cognito (user vs identity
  pools, social + SAML federation), DynamoDB (TTL, Streams, DAX, transactions), X-Ray
  (daemon, sampling, annotations), CI/CD, and containers (ECS, ECR, EKS, Fargate).
- Weighted to match the official blueprint:
  - Development with AWS Services (32%)
  - Security (26%)
  - Deployment (24%)
  - Troubleshooting & Optimization (18%)
- **Practice mode** (instant feedback + explanation per question) and **Exam mode**
  (answer straight through, scored at the end).
- Running timer, shuffled questions/answers, a per-domain mastery breakdown against the
  ~72% (720/1000) pass line, and a review of every missed question.
- Light/dark theme aware.

> Revision aid only. Not affiliated with or endorsed by Amazon Web Services. Pair it with
> the official AWS exam guide and hands-on practice.
